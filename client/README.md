# Smart City Monitoring Dashboard

A modern **Smart City Monitoring System** that allows users to monitor city services such as **Air Quality, Traffic, Transport, Announcements, and Emergency Services** through an interactive dashboard.

The project includes **authentication (Login/Register)** and a **responsive dashboard UI** with real-time style data visualization.

---

## Features

### Authentication

* User Registration
* Secure Login
* Password visibility toggle
* User session using **localStorage**
* Redirect after login/register

### Dashboard

* Personalized greeting (`Welcome <username>`)
* Interactive service cards
* Responsive UI
* Smooth hover animations

### Smart City Modules

* **Air Quality Monitoring**
* **Traffic Monitoring**
* **Public Transport Tracking**
* **City Announcements**
* **Emergency Services**

### UI / UX Improvements

* Modern gradient background
* Glassmorphism cards
* Responsive layout
* Clean typography
* Interactive flashcards

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/smart-city-dashboard.git
cd smart-city-dashboard
```

---

### 2. Install Backend Dependencies

```bash
cd backend
npm install
```

---

### 3. Start Backend Server

```bash
npm start
```

Backend runs on:

```
http://localhost:5000
```

---

### 4. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

### 5. Start Frontend

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
